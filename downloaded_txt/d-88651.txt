basic guide to fusing glass



learn how to fuse glass online. in this
        project you'll learn to fuse a 4 inch tile.
in order to fuse glass you have to know at least some basic properties of the
        glass. you'll learn that and more.
we also have instructions on how to make a fused dichroic glass
        pendant.      
    

                   
    

how to make marbles         
    

learn how to make glass marbles online. in this project you'll learn how to
        make a basic glass marble using a map gas torch. (one that uses propane or brazing fuel), or a bench
        torch that uses propane and oxygen.
