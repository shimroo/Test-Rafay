guide d'utilisation 
        / user guide




menu 
              principal / main menu






 
              - "n/n"
lance la configuration 
              des paramètres (taille du go-ban, handicap et komi) d'une nouvelle 
              partie. appel du menu "new".
to configure a new game (go-ban size, handicap and komi). call 
              of "new" menu.
  
          

 
              - "l/l"
pour charger 
              une partie au format sgf (ou mgt) sur le web ou sur votre disque 
              dur. appel du menu "load".
to load a game in sgf (or mgt) format on the web or on your hard 
              disk. call of "load" menu.
  
          



 
              - "s/s"
pour sauver 
              une partie au format sgf sur votre disque dur. selectionner le texte 
              et copier le vers votre éditeur préféré. puis sauver le avec une 
              extension ".sgf", il est alors chargeable par javago... 
              
to save a game in sgf format on your hard disk. select text to 
              be copied to your favorite editor. and save it with an ".sgf" 
              extension, it is then loadable with javago... 
  
          

 
              - "p/p"
passe. sur le 
              deuxième pass : comptage du score , clicker sur les chaines mortes 
              pour les supprimer, reclicker sur pass pour arrêter le comptage. 
              
pass. on second pass : scoring, click on dead chains to remove 
              them, reclick on pass to stop scoring.
  
          



 
va au premier 
              coup d'une partie.
go to first move of a game. 
  
          

 
              - "<"
recule d'un 
              coup dans une partie.
go one move before in a game. 
  
          



 
              - ">" or "w/w"
avance d'un 
              coup dans une partie.
go one move ahead in a game. 
  
          

 
              - ">|"
va au dernier 
              coup d'une partie.
go to last move of a game. 
  
          



 
              - ">#"
va au coup "n" 
              d'une partie.
go to move "n" of a game. 
  
          

 
              - "x/x"
supprime les 
              contrôles pour laisser le maximum de place au goban.
remove controls to leave maximum space to th goban.
  
          



  
              - "o/o"
détache/attache 
              l'applet à la page.
detach/attach the applet on the page.
  
          

flèches / arrows
déplace le curseur, 
              avec "shift" passe au dessus des pierres.
move the keyboard cursor, with "shift" pass over the 
              stones.

téléchargement / 
        download
télécharchez javago 
        / javago.zip (43ko-v0.17) / download 
        javago
après téléchargement, 
        il suffit d'extraire les fichiers de javago.zip dans un répertoire, puis 
        d'ouvrir le fichier javago.html dans votre browser compatible java. 
        dans certains cas (vieux browsers) il faut affecter la variable classpath 
        avec le nom de ce répertoire et rebooter la pc (ajouter la ligne "set 
        classpath=c:\repertoire\javago" dans le fichier "c:\autoexec.bat").
after download, you only have to extract files from javago.zip to one 
        directory, 
and open javago.html in your java compatible browser. in some case 
        (old browser) you have to set your classpath variable to this directory 
        and then reboot your pc (add this line: "set classpath=c:\path\javago" 
        in "c:\autoexec.bat" file ).
