faculty website template:
information technology provides a basic template for faculty to use to create new spectrum websites. this template is provided as a convenience and is not supported. two versions of the template are provided in compressed zip format. one may be used with any html site editor. the other package supports dreamweaver templates. the templates use html5 and offer responsive features that make them suitable for both desktop and mobile viewing. additional information regarding customization is available in the included readme.html file.
