desktop manager 





diese software wurde seit vielen jahren nicht weiter entwickelt.
desktop manager ein projekt zur implementierung eines einfach zu bedienenden virtuellen desktop-managers für os x. starten sie einfach die app und ein pager sollte in der statusleiste und auf dem desktop erscheinen. die version 0.5.3 ist die stabilste version und wird für neue benutzer empfohlen. es ist os x tiger-kompatibel, aber es wird keine weitere entwicklung für diesen zweig durchgeführt. wenn sie bereit sind, unvollständigen code auszuprobieren, können sie den neuesten entwickler-build ausprobieren. dies kommt von dem code, der 0.6 bilden wird, aber noch nicht vollständig ist und fehler aufweisen kann. merkmale

beliebige anzahl von benannten virtuellen ‚bildschirmen‘ (bis zum verfügbaren speicherplatz), auf denen sie ihre programme anordnen können. haben sie einen bildschirm, auf dem sie ihre e-mail-programme aufbewahren können, einen für das surfen im web und einen für die arbeit. die möglichkeiten sind fast unbegrenzt.
spannende schalterübergänge, um alle ihre windows/linux-anwendungen mit freunden vor neid erblassen zu lassen.
steuern sie das schalten über voll konfigurierbare „hotkeys“.
verschaffen sie sich einen sofortigen überblick über ihre desktops über einen auf der statusleiste basierenden „pager“.
verschieben sie fenster auf desktops über einen desktopbasierten „pager“.
desktop-pager, der mit codetek virtualdesktop-skins kompatibel ist.
verschieben sie fenster zwischen den desktops einfach über das menü in der statusleiste oder expose-ähnlich dem’operationsmenü‘.
unterstützung für klebrige fenster – haben sie ein fenster auf allen desktops.
aktive kantenunterstützung – wechselt den desktop, wenn sich die maus über den bildschirmrand bewegt.
lünette für desktop-benachrichtigungen – erscheint auf dem desktop-switch, um ihnen bei der orientierung zu helfen.
aktueller desktop-name, der auf dem desktop-hintergrund verfügbar ist, damit sie den überblick behalten.
kleine, unauffällige app – kann so gestaltet werden, dass sie völlig unsichtbar ist, wenn sie wollen. entwickelt, um ihnen aus dem weg zu gehen und sich wie ein teil des betriebssystems zu verhalten.
