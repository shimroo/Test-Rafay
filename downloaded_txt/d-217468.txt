the r environment
r is an environment associated with statistical computing and data analysis. it is an environment that offers its own programming language, memory management, and a number of libraries to extend the features available. the r language is a free implementation of the s programming language. it can be found at https://www.r-project.org/.


matlab
matlab is an environment associated with scientific computing and numerical linear algebra. it is an environment that offers its own programming language, memory management, and a number of libraries to extend the features available. matlab is a commerical software product, and more information can be found at https://www.mathworks.com/. a free implementation of the matlab language can be found at https://www.gnu.org/software/octave/.
