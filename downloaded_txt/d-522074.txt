accueil - réservation

le parc et le château sont accessibles sans réservation préalable pour les individuels.
attention, en raison de travaux importants de restauration, le parcours de visite risque d'être impacté durant l'année 2025-2026. 
veuillez nous excuser de la gêne occasionnée.
réservation obligatoire pour les visites de groupes, avec ou sans guide (pas de groupe accepté le 1er dimanche du mois) : reservation.malmaison@culture.gouv.fr  
pour les autres réservations (conférences, visites guidées, ateliers, concerts) nous vous invitons à cliquer sur le lien suivant : réserver
pour acheter votre billet en ligne nous vous invitons à cliquer sur le lien suivant : acheter 






horaires - tarifs

ouverture tous les jours sauf le mardi.
château de malmaison (collections permanentes) :
tarif : plein 6,5€ / réduit : 5 € / groupe : 5,5€ / parc seul : 1,5€
château de bois-préau (exposition temporaire) :
tarif : plein 6,5€ / réduit : 5€ / groupe : 5,5€
billet jumelé :
tarif : plein 11 € / réduit : 8€ / groupe : 8,5€
les billets achetés en ligne font l'objet d'une commission de 1€50 en sus du prix d'entrée.
